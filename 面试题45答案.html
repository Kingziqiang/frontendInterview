	  2、min()和max()方法
	     2.1 min()获取最小值
	         2.1.1 参数：数字类型，如果是非数字类型会转成数字类型，然后进行比较
	         2.2.2 返回值:如果全部是数字类型，则返回最小值。
                          如果包含非数字类型，则将非数字类型强制转成数字类型进行比较。
                          如果非数字类型无法转换有效数字，则返回NaN
             Math.min(1,2,3) //1
             Math.min(1,2,3,'a') //NaN
             Math.min(1,2,3,false) //0
             Math.min(2,3,true) //1
             Math.min(1,2,3,[]) //0
             Math.min(1,2,3,null) //0
             Math.min(1,2,3,undefined) //NaN
             Math.min(1,2,3,{}) //NaN
	     2.2 max()获取最大值
	         2.1.1 参数：数字类型，如果是非数字类型会转成数字类型，然后进行比较
	         2.2.2 返回值:如果全部是数字类型，则返回最大值。
                          如果包含非数字类型，则将非数字类型强制转成数字类型进行比较。
                          如果非数字类型无法转换有效数字，则返回NaN
             Math.max(1,2,3) //3 
             Math.max(1,2,3,'a') //NaN
             Math.max(-1,-2,-3,false) //0
             Math.max(-2,-3,true) //1
             Math.max(-1,-2,-3,[]) //0
             Math.max(-1,-2,-3,null) //0
             Math.max(1,2,3,undefined) //NaN
             Math.max(1,2,3,{}) //NaN

         3.1 Math.ceil()
             执行向上舍入，如果是小数且小数点后包含非零数字则取相邻的两个整数中最较大值，如果是整数或小数点后全是0则取整数本身。

             Math.ceil(1.4)  //2
             Math.ceil(1.5)  //2
             Math.ceil(1.6)  //2
             Math.ceil(2.0)  //2
             Math.ceil(-1.0) //-1
             Math.ceil(-1.4) //-1
             Math.ceil(-1.5) //-1
             Math.ceil(-1.6) //-1
             Math.ceil(-2.0) //-2

      1 字符串的特性
        1.1 length
            var str="hello world";
            console.log(str.length); //11
            var str="你好";
            console.log(str.length); //2 通过length属性获取字符串长度时，一个双字节字符(比如汉字)也只算一个字符
        1.2 下标
            var str="hello world";
            console.log(str[4]); //o 第4个下标(第5个字符)为字母o
        1.3 字符串是不可变的。
            字符串一旦创建，它们的值就不能改变。
            要改变某个变量保存的字符串，首先要销毁原来的字符串，然后用另一个包含新址的字符串填充该变量。
            var str="a";
            console.log(str); //"a"
            str+="b";
            console.log(str); //"ab"
            虽然变量保存的字符串变了，但原的字符串"a"没变。
      2 字符方法
        2.1 charAt(index)
                  参数：index表示字符串索引，索引从0开始
                  返回值：位置index处的字符
                  原变量存储的字符串是否发生改变：不变
                  var str ="hello abc";
                  str.charAt(6); //"a"  
                  console.log(str);  //"hello abc"         
        2.2 charCodeAt(index)
                  参数：index表示字符串索引，索引从0开始
                  返回值：位置index处的字符编码
                  原变量存储的字符串是否发生改变：不变
                  var str ="hello abc";
                  str.charAt(6); //97  "a"的十进制ascii码为97
                  console.log(str);  //"hello abc"
      3 字符串操作方法
        3.1 concat(x)
                  参数：x表示拼接的字符串
                  返回值：原字符串和x拼接后的字符串
                  原变量存储的字符串是否发生改变：不变  
			      var str="a";
			      str.concat("b"); //"ab"
			      console.log(str);  //"a"
        3.2 slice(n,m)
                  参数：n表示获取字符串的起始字符串，m表示获取字符串的结束字符串,包含n不包含m
                  返回值：如果n和m都是正数，返回字符串第n到m-1个字符。
                          如果n是正数m是负数，返回字符串第n到第m-1+length个字符。
                          如果n是负数m是正数，返回字符串第n+length到第m-1个字符。
                          如果n和m都是负数，返回字符串第n+length到第m-1++legnth个字符
                  原变量存储的字符串是否发生改变：不变  
			      var str1="0123456789";
			      str.slice(4,7); //"456"
			      console.log(str);  //"0123456789"
			      str.slice(4,-3); //"456" 等价于str.slice(4,-3+str.length)等价于str.slice(4,7)
			      str.slice(-6,7); //"456" 等价于str.slice(-6+str.length,7)等价于str.slice(4,7) 
			      str.slice(-6,-3); //"456" 等价于str.slice(-6+str.length,-3+str.length)等价于str.slice(4,7) 
	      3.3 substring(n,m)
                  参数：n表示获取字符串的起始字符串，m表示获取字符串的结束字符串,包含n不包含m
                  返回值：如果n和m都是正数，返回字符串第n到m-1个字符。
                          如果n是正数m是负数，返回字符串第0到第n-1个字符。
                          如果n是负数m是正数，返回字符串第0到第m-1个字符。
                          如果n和m都是负数，返回空字串""
                  原变量存储的字符串是否发生改变：不变  
			      var str="0123456789";
			      str.substring(4,7); //"456"
			      console.log(str);  //"0123456789"
			      str.substring(4,-3); //"0123" 等价于str.substring(0,4)
			      str.substring(-6,7); //"0123456" 等价于str.substring(0,7)
			      str.substring(-6,-3); //"" 等价于str.substring(0,0) 
		  3.4 substr(n,m)
                  参数：n表示获取字符串的起始字符串，m表示获取字符串的长度
                  返回值：如果n和m都是正数，返回字符串第n到m-1个字符。
                          如果n是正数m是负数，返回空字符串""。
                          如果n是负数m是正数，返回字符串第n+length个位置开始取m个字符。
                          如果n和m都是负数，返回空字串""
                  原变量存储的字符串是否发生改变：不变  
			      var str="0123456789";
			      str.substr(4,7); //"4566789"
			      console.log(str);  //"0123456789"
			      str.substr(4,-3); //"0123" 等价于str.substr(0,4)
			      str.substr(-6,7); //"456789" 等价于str.substr(-6+str.length,7)等价于str.substr(4,7) 
			      str.substr(-6,-3); //"" 等价于str.substr(0,0) 
	  4 字符串位置方法
	      4.1 indexOf()
			    参数：两个参数、数字类型，indexOf(x,n)代表从索引为n处开始查找x，n为可选参数，当省略n时代表从字符串第0个字符开始查找x。
                返回值：从字符串第n个字符从开始往后查找，x在字符串中第一次出现的位置。
                原字符串是否发生改变：不变
                 		     
          4.2 lastIndexOf()
			    参数：两个参数、数字类型，lastIndexOf(x,n)代表从索引为n处开始查找x，n为可选参数，当省略n时代表从字符串第0项开始查找x。
                返回值：从字符串第n项开始往后查找，x在字符串中跟第一次出现的位置。
                原字符串是否发生改变：不变

       7 字符串的模式匹配方法
           7.1 match()
                参数：正则。
                返回值：返回一个数组，数组第一项是正则匹配的字符串
                原字符串是否发生改变：不变
                var str="cake wake";
                var reg=/[a-z]ake/
                var matches = str.match(reg);
                console.log(matches); //["cake", index: 0, input: "cake wake"]
                console.log(str);  //"cake wake"
           7.2 search()
                参数：正则。
                返回值：返回字符串第一个匹配项的索引，如果没有找到匹配项则返回-1。
                原字符串是否发生改变：不变
                var str="cake wake";
                var reg1=/ake/;
                var reg2=/abc/;
                str.search(reg1); //1
                str.search(reg2); //-1
                console.log(str);  //"cake wake"
           7.3 replace(r,f)
                参数：第一个参数是正则或者字符串，第二个参数是函数或字符串。
                返回值：返回被替换后的字符串。
                原字符串是否发生改变：不变
                var str="wife knife";
                str.replace("fe","ves"); //"wives knife";
                var reg=/fe/g;                
                str.replace(reg,"ves"); //"wives knives";               
                function fn(match){
                    return "ves";
                }
                str.replace(reg,fn); //"wives knives";  
                console.log(str);  //"cake wake"
            7.4 split(r,n)
                参数：第一个参数可以是正则或字符串，第二个参数用来指定数组的长度。
                返回值：返回一个数组，按指定的字符串或正则拆分字符串存入数组
                原字符串是否发生改变：不变
                var str="cake,wake,make";
                str.split(); //["cake,wake,make"],将整个字符串作为数组的第一个元素存入数组，数组的长度为1
                str.split(","); //["cake", "wake", "make"]
                str.split(",",2); //["cake", "wake"]
                var reg=/ake/
                str.split(reg); //["c", ",w", ",m", ""]                
                console.log(str);  //"cake,wake,make"
      8 localeCompare();
                参数：被比较的字符。
                返回值：如果字符查相等，则返回0.
                        如果字符串字符编码小于字符串参数则返回-1；
                        如果字符串字符编码大于字符串参数则返回1；
                原字符串是否发生改变：不变
                var str="coat";
                str.localeCompare("coat"); //0
                str.localeCompare("cake"); //1    
                str.localeCompare("make"); //-1           
                console.log(str);  //"coat"
